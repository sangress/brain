# brain — AI-powered Bash command completion

Turn natural language into shell commands directly in your Bash prompt.

Type what you want → press Ctrl+G → command appears (not executed).

## Usage
Type:
show git config username
Press Ctrl+O

## Requirements
- bash
- python3
- OpenAI API key

## Safety
Commands are NOT auto-executed.
